CEDDet: A Cloud–Edge Collaborative System for Real-Time Driver Distraction Detection in Low-Resource Scenarios

Abstract

Driver distraction detection is critical to road safety. However, the real-world deployment of distraction detection models is often hindered by limited on-board computational resources. It is also constrained by the scarcity of labeled data. To address these challenges, we propose CEDDet, A Cloud-Edge collaborative system for real-time Driver Distraction Det}ection in low-resource scenarios. On the edge side, we design a lightweight distracted-frame mining module for key-frame selection. The selected frames are then uploaded to the cloud. In the cloud, we introduce an automated annotation pipeline that integrates a strong pose detector with a Vision-Language Model (VLM). It annotates the uploaded key frames, thus constructing a high-quality training dataset without any human involvement. Using the automatically built dataset, CEDDet performs closed-loop iterative training to produce a lightweight object detector. The weights are periodically sent back to the edge for real-time inference, while large models remain cloud-only as offline auxiliary tools. Experiments show that CEDDet achieves good performance while meeting real-time device constraints. Even when the data distribution shifts, the lightweight object detector obtained via iterative closed loop training maintains stable and usable detection accuracy, demonstrating strong generalization and transferability.
